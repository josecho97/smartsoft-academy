Curso: HTML desde cero

Descripción más completa

Universidad HTML. En este curso aprenderás todo lo referente a HTML (Hypertext Marketing Language) para crear sitios y aplicaciones Web increíbles. Desarrollaremos múltiples aplicaciones para poner en práctica todos los conceptos que estudiaremos a lo largo del curso. No necesitas tener ningún conocimiento previo, ya que comenzaremos desde cero. Para la creación de aplicaciones y sitios Web es necesario conocer las tecnologías de HTML, CSS y JavaScript. Por lo que los temas de este curso lo puedes estudiar también en el curso de Universidad Desarrollo Web donde se enseñan las 3 tecnologías HTML, CSS y JavaScript. Sin embargo, si deseas aprender sólo HTML puedes adquirir este curso, además en un futuro agregaremos temas únicos a este curso, que no se encontrará en ningún otro curso. Así que de momento si ya adquiriste el curso de Universidad Desarrollo Web NO es necesario que adquieras este curso.


Que aprenderás:

  • Crear aplicaciones Web de Cero a Experto, utilizando HTML con las últimas versiones al día de hoy!
  • HTML es la base para la creación de aplicaciones Web, por lo que se requiere para entender muchas otras tecnologías, este curso es la base para muchos cursos!
  • Aprenderán desde cero HTML con el objetivo de crear increíbles aplicaciones Web
  • Crearemos múltiples aplicaciones Web para poner en práctica todos los temas estudiados en el curso!

Conocimientos previo:

  • No se requiere ningún conocimiento previo, empezaremos desde cero!
